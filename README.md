# Install
```
cd $HOME
git clone https://github.com/beko11/SUPER.git
cd SUPER
chmod +x install.sh
chmod +x launch.sh
chmod +x tg
./install.sh
./launch.sh
# Enter a phone number & confirmation code.
```
Create a bot!
